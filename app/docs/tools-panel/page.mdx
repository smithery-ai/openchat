# Tools Panel

The ToolsPanel component displays a searchable grid of tools from a connected MCP server. It's designed to be used within a connection context.

<ToolsPanelPreview />

## Installation

<InstallCommand name="smithery-tools-panel" />

## Usage

```tsx
import { ToolsPanel } from "@/components/smithery/tools-panel"

// Tools come from an MCP client connection
const tools = await mcpClient.tools()

export function MyApp() {
  const handleExecute = async (toolName: string, params: Record<string, unknown>) => {
    const tool = tools[toolName]
    return await tool.execute(params, { toolCallId: "manual", messages: [] })
  }

  return (
    <ToolsPanel
      tools={tools}
      onExecute={handleExecute}
    />
  )
}
```

## Props

- `tools` (required): A record of tools from the AI SDK, typically from `mcpClient.tools()`
- `onExecute` (optional): Async function to execute a tool. Receives the tool name and parameters

## Features

- Searchable grid layout of tool cards
- Filter tools by name, description, or type
- Responsive grid (1-3 columns based on screen size)
- Shows tool count and filtered count
- Each tool opens a detail dialog on click
- Empty states for no tools or no search matches

## Integration

The ToolsPanel works best when wrapped with a `ConnectionConfigContext.Provider` to enable code generation in the tool detail dialog:

```tsx
import { ConnectionConfigContext } from "@/components/smithery/connection-context"

<ConnectionConfigContext.Provider value={{
  mcpUrl: "https://mcp.exa.ai",
  apiKey: "your-token",
  namespace: "your-namespace",
  connectionId: "your-connection-id"
}}>
  <ToolsPanel tools={tools} onExecute={handleExecute} />
</ConnectionConfigContext.Provider>
```
