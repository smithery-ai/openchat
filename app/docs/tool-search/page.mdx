# Tool Search

The ToolSearch component provides a global search interface for finding tools across all your connected MCP servers.

<ToolSearchPreview />

## Installation

<InstallCommand name="smithery-tool-search" />

## Usage

```tsx
import { ToolSearch } from "@/components/smithery/tool-search"

export function MyApp() {
  return (
    <ToolSearch token="your-smithery-api-token" />
  )
}
```

## Props

- `token` (required): Your Smithery API token for authentication

## Features

- Search across all tools from all connected servers
- Debounced search for optimal performance
- Shows tool name, description, and source server
- Expandable input schema preview
- Integrated ServerSearch for adding new servers when tools aren't found
- Loading and empty states with helpful guidance
