# Tool Card

The ToolCard component displays an individual MCP tool as a clickable card. Clicking the card opens a detail dialog where users can view parameters and execute the tool.

<ToolCardPreview />

## Installation

<InstallCommand name="smithery-tool-card" />

## Usage

```tsx
import { ToolCard } from "@/components/smithery/tool-card"

// A single tool from an MCP client
const tool = tools["search"]

export function MyApp() {
  const handleExecute = async (params: Record<string, unknown>) => {
    return await tool.execute(params, { toolCallId: "manual", messages: [] })
  }

  return (
    <ToolCard
      name="search"
      tool={tool}
      onExecute={handleExecute}
    />
  )
}
```

## Props

- `name` (required): The name of the tool to display
- `tool` (required): The AI SDK Tool object with description and schema
- `onExecute` (optional): Async function to execute the tool. Receives the parameters

## Features

- Displays tool name, description, and type badge
- Hover effect with chevron indicator
- Click to open the ToolDetailDialog
- Truncates long descriptions to 2 lines
- Responsive card that works in grid layouts
